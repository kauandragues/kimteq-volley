# kimteq-volley
KIMTEQ VOLLEY é uma sistema feito em C que mostra de forma visual as posições e situações do voleibol para ensinar os alunos de forma didática.

## Como compilar o programa
- Vá até a pasta src
- Execute o comando:
``gcc main.c -o ../simulador.exe```
- Um simulador.exe deve ter sido criado na pasta principal do projeto
